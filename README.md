Projeto Sistema de Alerta de Secas do Estado do Rio de Janeiro - SAS/RJ
www.sasrj.com.br

O site tem por finalidade facilitar a visualização dos dados de SPI/SPEI disponibilizados pelo INPE, facilitando o monitoramento do estado de seca dos municípios do estado do Rio de Janeiro. Isso é feito através de um mapa interativo, que possibilita que o usuário analise a situação geral do estado, e emite alertas para áreas do estado em situação crítica. Gera também relatórios mensais, que são enviados por e-mail aos usuários inscritos.
